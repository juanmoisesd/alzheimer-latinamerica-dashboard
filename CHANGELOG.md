## [1.0.0] - 2025-01-01

### Added
- Initial release of the Alzheimer Latin America Dashboard dataset
- CITATION.cff for standardized academic citation
- CONTRIBUTING.md with contribution guidelines
- CODE_OF_CONDUCT.md (Contributor Covenant v2.1)
- SECURITY.md with vulnerability disclosure policy
- NOTICE file with data source attribution
- robots.txt and sitemap.xml for SEO
- PWA manifest.json for mobile support
- OpenAIRE-compliant .zenodo.json metadata

### Data
- Complete epidemiological dataset 2000–2024
- 20+ countries covered
- Multi-format exports (CSV, JSON)
