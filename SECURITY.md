# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it
responsibly by emailing **jmsernatuya@gmail.com** with the subject line:
`[SECURITY] Alzheimer Latin America Dashboard`.

**Please do NOT open a public GitHub issue for security vulnerabilities.**

We will acknowledge your report within 48 hours and provide a detailed response
within 7 days, including an estimated timeline for a fix.

## Scope

This repository contains research datasets and dashboards. Security concerns include:
- Data integrity issues that could lead to misleading research conclusions
- Privacy violations in any linked or derived data
- Malicious content injected via third-party dependencies

## Out of Scope
- Issues with GitHub's own platform
- Data accuracy disputes (please open a regular issue for those)
