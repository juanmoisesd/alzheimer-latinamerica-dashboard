# Reproducibility Guide

## Dataset: Interactive dashboard: Alzheimer's Disease in Latin America (2000-2050). 8 countries, projections, policy index, caregiver burden. DOI: 10.7910/DVN/UVHABW

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/alzheimer-latinamerica-dashboard.git
cd alzheimer-latinamerica-dashboard
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
